概述

FLV Parser 是一个用于解析和编辑 FLV 视频文件的工具，支持查看tag详情、删除tag以及修改二进制数据。

FLV解析功能

打开文件

  • 点击工具栏的 打开 按钮，或使用菜单 文件 → open
  • 选择要解析的 FLV 文件
  • 文件解析完成后，左侧会显示所有tag的列表

查看tag信息

  • 在左侧 tag列表 中点击任意一行，右上方 tag信息树 会显示该tag的详细字段结构，右下方 二进制数据表 会以十六进制格式显示tag的原始字节
  • 在右上方 tag信息树 中点击某个字段，二进制数据表会自动高亮对应的字节范围

FLV编辑功能

⚠️ 注意事项：此操作直接修改源文件，无法撤销！建议操作前备份原文件

删除FLV Tag

  1. 在左侧 tag列表 中选中要删除的tag
  2. 右键点击 该tag，在弹出的菜单中选择 删除tag
  3. 在确认对话框中点击 Yes
  4. 文件会立即被修改，并自动重新加载

修改二进制字节

  1. 在右上方 tag信息树 中点击某个字段，二进制数据表会自动高亮对应的字节范围
  2. 在右下方 二进制数据表 中，双击 要修改的单元格
  3. 输入新的十六进制值（范围：00-FF，不区分大小写）
  4. 按 Enter 键确认
  5. 确认后，文件会立即被修改并自动重新加载所有视图

查看日志

  • 使用菜单 查看 → 查看日志 或点击工具栏的日志图标
  • 可以按日志级别筛选（全部、Debug、Info、Warning）
